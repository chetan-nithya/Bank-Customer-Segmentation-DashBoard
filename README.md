# Bank-Customer-Segmentation-DashBoard
This project focuses on analyzing banking data to understand customer segmentation and transaction patterns.

# Project Overview
This project focuses on analyzing banking data to understand customer segmentation and transaction patterns. The dashboards help visualize customer distribution, financial activity, and key metrics to support data-driven decision making.

# Customer Segmentation Dashboard
The first dashboard analyzes customer characteristics and financial distribution.
Key Metrics
* Total Balance: 119.82bn
* Total Customers: 880K
* Total Transactions: 1M
* Average Transaction Value: 1.57K
* High Value Customers: 173K

# Insights
* Customers are divided into Low Value, Medium Value, High Value, and Premium segments
* Premium customers hold the largest share of total balance
* The 36–50 age group has the highest number of customers
* Gender distribution shows higher participation from male customers
* Average balance varies across different age groups
* Transaction Analytics Dashboard

The second dashboard focuses on analyzing customer transaction behavior.
# Key Metrics
* Total Transactions: 1M
* Total Customers: 880K
* Total Transaction Amount: 1.63bn
* Average Balance: 114.90K
* Transactions per Customer: 1.18

# Transaction activity shows monthly variations
* Top 10 customers contribute a significant share of transaction volume
* Transaction distribution differs across customer segments
* Male customers perform more transactions compared to other genders
* Balance and transaction amount show correlation patterns
